# SMS Trainer

## Overview
- This projects purpose is to serve as a clone of the SMS that is loaded into the GCS cockpits for the purposes of training. This project provides aircrew and ground control station operators with realistic training scenarios for weapons systems management, payload configuration, and mission planning procedures.

## Getting started
- You'll need to download Git and have a text editor available
- To run:
    - git clone repo
    - cd sms-trainer
    - `npm install`
    - `ng serve`

## Key Features
Interactive Stores Configuration Interface : Simulate loading, configuring, and managing various payload types including sensors, weapons, and fuel tanks
Mission Planning Integration : Practice pre-flight stores planning and configuration validation
Systems Integration Training : Learn interactions between stores management and other aircraft systems
